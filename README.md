### Shortcuts table :

| Shortcuts | Description |
| :---: | :---: |
| `ctrl + r` | **Reload** page |
| `ctrl + ,` | Opens the **settings** popup |
| `<--` | Is used to navigate to the **previous** page |
| `-->` | Is used to navigate to the **next** page |
